# Modern DSA Tutorial - Interactive Learning Platform

Welcome to the **Modern DSA Tutorial**! This is an interactive learning platform designed to help you master Data Structures and Algorithms. The platform is powered by the Google Gemini API to provide intelligent explanations and guidance.

## Features

*   **Interactive Learning:** Engaging tutorials on various DSA topics.
*   **AI-Powered Explanations:** Get deep insights and help with concepts using the integrated Gemini API.
*   **Modern UI:** A clean and responsive user interface built with HTML, CSS, and vanilla JavaScript.
*   **Fast Development:** Uses Vite for quick and optimized local development.

## Technologies Used

*   HTML5
*   CSS3
*   Vanilla JavaScript
*   [Vite](https://vitejs.dev/) - Frontend Tooling
*   [Google Gemini API](https://ai.google.dev/) - AI Integration

## Getting Started

Follow these steps to run the application locally on your machine.

### Prerequisites

You will need to have Node.js and npm installed.

### Installation

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone https://github.com/Arju1234n/Dsa-totur.git
    cd "Dsa-totur"
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up your environment variables:**
    *   Create a `.env` file in the root directory (or use the existing one).
    *   Add your Google Gemini API key:
        ```env
        VITE_GEMINI_API_KEY=your_actual_api_key_here
        ```
    *   *Note: Never commit your `.env` file to version control.*

4.  **Start the development server:**
    ```bash
    npm run dev
    ```

5.  **Open the application:**
    Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173/`).

## Project Structure

*   `index.html`: The main entry point for the application.
*   `style.css`: Contains all the styling for the platform.
*   `script.js`: Handles the interactive logic and API calls.
*   `package.json`: Manages project dependencies and scripts.
*   `.env`: Stores environment variables (like API keys).
